Follow these steps to upload your project to GitHub with a complete profile bio.

---

## STEP 1 — Create GitHub Account (skip if you have one)
1. Go to https://github.com
2. Click **Sign up**
3. Enter username, email, password
4. Verify your account

---

## STEP 4 — Upload via Git (Terminal / VS Code)

### Install Git if needed
```bash
# macOS
brew install git

# Ubuntu/Debian
sudo apt install git

# Windows — download from https://git-scm.com
```

### Configure Git (first time only)
```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

### Initialize and Push
```bash
# Navigate to the project folder
cd bengaluru-house-price-prediction

# Initialize git
git init

# Add all files
git add .

# First commit
git commit -m "🏠 Initial commit: Bengaluru House Price Prediction ML project

- Complete Jupyter Notebook with EDA, cleaning, modelling
- 6 ML models compared (Random Forest selected as best)
- Training script and CLI prediction utility
- README with badges, instructions, model results"

# Add remote origin (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/bengaluru-house-price-prediction.git

# Push to GitHub
git branch -M main
git push -u origin main
```

---

## STEP 5 — Upload via VS Code (GUI method)

1. Open VS Code
2. Open the project folder: **File → Open Folder** → select `bengaluru-house-price-prediction`
3. Click the **Source Control** icon (left sidebar, looks like a branch)
4. Click **Initialize Repository**
5. Stage all files: click **+** next to "Changes"
6. Enter commit message: `🏠 Initial commit: Bengaluru House Price Prediction`
7. Click **✓ Commit**
8. Click **Publish Branch**
9. Sign in to GitHub when prompted
10. Choose **Public** repository

---

## STEP 6 — Add Repository Topics (Tags)
